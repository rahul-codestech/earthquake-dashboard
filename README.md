# 🌍 Global Earthquake Dashboard

Interactive earthquake analytics dashboard built using Python and Plotly. The dashboard fetches real-time earthquake data from the USGS Earthquake API and provides visual insights through maps, charts, and trend analysis.

## ✨ Features

- Real-time earthquake data from USGS API
- Global earthquake map visualization
- Magnitude distribution analysis
- Magnitude vs Depth correlation
- Daily earthquake trend tracking
- Top active earthquake regions
- Interactive charts and filters

## 🛠️ Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
- Requests
- USGS Earthquake API

## 📊 Insights Generated

- Total earthquakes in the last 30 days
- Strongest earthquake recorded
- Average earthquake magnitude
- Tsunami alert statistics
- Region-wise earthquake activity
- Historical trend analysis

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rahul-codestech/earthquake-dashboard.git
cd earthquake-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 🎯 Project Goal

To demonstrate data collection, data analysis, API integration, and interactive dashboard development using Python.
